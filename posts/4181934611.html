Navicat Premium 15 linux 安装与激活 ArchLinux 2022

Created2021-04-12|Updated2022-07-21|Software
|Word count:2.2k|Reading time:12min|Post View:

不支持 Navicat16!!!
Navicat15.0.30 链接: https://pan.baidu.com/s/1Oe8uSbQ6F8e2cglrra8wNw 提取码: nwrk
参考https://github.com/lzscxb/navicat-keygen

下载地址

http://www.navicat.com.cn/download/navicat-premium

提取AppImage文件到文件夹里

cp的时候可能比较久请耐心等待 尽量目录一致

安装 Navicat Keygen

前提条件

你的gcc支持C++17特性

建议安装 gcc11

安装 capstone

安装 keystone

方式一. aur仓库（推荐）

方式二. 编译安装(不推荐)

编译安装（这里时间比较久，耐心等待会…）

注意：使用编译安装会存在 libkeystone.so.0 找不到的问题需要链接到 /lib

编译安装

生成完成后，你会在 ./bin 文件夹下看到编译后的keygen/patcher。

navicat-patcher 替换官方公钥

样例输出

重新打包成 AppImage

安装 appimagetool

yay 可能需要网络代理才能下载
也可以在 github 下载 https://github.com/AppImage/AppImageKit/releases

打包

运行 Navcat 15

Navicat-keygen 生成序列号和激活码

你会被要求选择Navicat产品类别、Navicat语言版本和填写主版本号。之后一个随机生成的 序列号。

需要断开网络连接，否则会激活失败

你可以使用这个 序列号 来暂时激活Navicat。

之后你会被要求填写 用户名 和 组织名。你可以随意填写，但别太长。

此时会进入等待状态 你会被要求填写请求码。
注意：不要关闭keygen。

断开网络. 找到注册窗口，填写keygen给你的 序列号，然后点击 激活

通常在线激活会失败，所以在弹出的提示中选择 手动激活。

复制 请求码 到keygen，连按两次回车结束。

最终你会得到一个base64编码的 激活码。

将之复制到 手动激活 的窗口，然后点击 激活。

如果没有什么意外，应该可以成功激活。

清理

可能会出现的问题

Try to open libcc.so … Not Found

Press ENTER to continue or Ctrl + C to abort.
[+] Try to open libcc.so … Not Found!

执行 sudo ./bin/navicat-patcher ../navicat 时 ../navicat 路径找不到

libcrypt.so.1: cannot open shared object

/tmp/.mount_navicaL06GBK/AppRun: error while loading shared libraries: libcrypt.so.1: cannot open shared object file: No such file or directory

/libgio-2.0.so.0: undefined symbol: g_module_open_full

/tmp/.mount_navicaa3IXPX/AppRun: symbol lookup error: /usr/lib/libgio-2.0.so.0: undefined symbol: g_module_open_full

glib2 >= 2.7.0 时 会出现此类问题
解决办法：将 glib2 2.68.4 集成到 navicat 中

  1. 下载 glib2 2.68.4
    https://archive.archlinux.org/packages/g/glib2/glib2-2.68.4-1-x86_64.pkg.tar.zst
  2. 解压并将 usr 复制的 navicat 中

libffi.so.7: cannot open shared object

/tmp/.mount_navica9bFtLj/AppRun: error while loading shared libraries: libffi.so.7: cannot open shared object file: No such file or directory

同上面的一样 将 libffi7 补丁打到 navicat 中

  1. 下载 libffi7
    https://packages.ubuntu.com/focal/libffi7
  2. 解压data.tar.xz 并将 usr/lib/x86_64-linux-gnud/* 复制的 navicat/usr/lib 中
